hardload
     Программа служит для копирования файла в выбранную папку.
Чтобы начать работать в программе, нужно подготовить файлы для их копирования и создать папки, куда будут скопированы файлы
Для работы в системе windows установка программы hardload не требуется, достаточно запустить файл hardload.exe

Кнопка "найти и добавить файл в список" позволяет найти нужнуй файл на вашем ПК и добавить в список 
кнопка "Удалить выделеные файлы из списка" удаляет тот файл который выбран чекбоксом (чалочкой)
кнопка "выбрать папку, куда писать файлы" при нажатии на ее, открывает окно с вобором папок, куда будет скопирован файл
кнопка "начать грузить" при нажатии на нее выбранный файл копируется в выбранную папку 
кнопка "остановить грузить" при нажитии на нее, процесс копирования приостанавливается 

     Необходимо добавлять по одному файлу в список
